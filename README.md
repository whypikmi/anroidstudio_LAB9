## Лабораторная работа №9. Создание собственного приложения на основе Jetpack Compose
### Цель
Создать полноценное Compose-приложение (фильмы / аниме / игры — на ваш выбор), используя знания всех предыдущих лабораторных работ: компонуемые элементы, состояние, обработку событий, работу с макетами, переключателями, кнопками, изображениями и поддержку ориентации экрана.
### Тема: Список любимых сериалов
#### Compose-интерфейс
Были использованы следующие элементы:
* Column / Row
* Image
* Text
* Surface
* выравнивание и отступы
* Modifier для размеров
#### Кнопки
Были добавлены две интерактивные кнопки, выполняющие:
* Переключение на следующее/предыдущее изображение.
* Переключение по кругу (последний → первый).
